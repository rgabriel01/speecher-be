# Speecher [Backend]

# Tech Stack
* Ruby 2.7.1
* Rails 7.0.6

## Setup
1. Clone the repository
```
$ git clone git@github.com:rgabriel01/speecher-be.git
$ cd speecher-be
```
2. Install dependencies
```
bundle install
```
3. Seed the database with initial Authors
```
rails db:seed
```
4. Run the server
```
rails s
```
